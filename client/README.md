# client

## Project setup
```
in server/
npm install
npm install -D nodemon
npm install cors mongodb body-parser express
npm run seeds
in client/
npm install
```

### to run during development
```
3 terminals required for running background services:
mongod
in server/
npm run server:dev
in client/
npm run serve
in web browser:
localhost:8080
```
